环环相扣

题目描述
完成函数bool isCircularLinkedList(Node* head)，判断输入的不带头结点的链表是否是有环。如果是，返回true，否则返回false。保证每个链表的长度至少为1。

输入样例
3
0 3 1 2 3
0 2 1 2
1 3 1 2 3
​
输出样例
false
false
true
​
提示
请记得将头文件包含进去，即#include "List.h"
有环的链表不一定符合“循环链表”的定义
本题有O(1)额外空间复杂度，O(n)时间复杂度的算法（本题设置的时间和空间限制较为宽松，一些更高时空复杂度的算法也可通过，但建议学有余力的同学思考更优的算法设计）
